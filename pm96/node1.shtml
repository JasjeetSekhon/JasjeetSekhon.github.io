Introduction

Cox (1962; 1961) proposed a fairly general method that can be used to test alternative hypotheses from separate statistical families. The Cox test can be interpreted as a classical procedure (Dastoor 1985), so that the familiar machinery of classical hypothesis testing can be relied upon when using the Cox test. White (1982) rigorously derived regularity conditions under which the Cox test is asymptotically normal, and Aguirre-Torres and Gallant (1983) used a bootstrap method to derive a distribution-free form of the test for nonlinear regression models. But Loh (1985) showed that notwithstanding asymptotic results, hypothesis tests based on the Cox statistic may not have the correct size.

Using a formulation of the statistic described by White (1994, 283), we use results from Beran and Srivastava (1985) and Hall (1992) to prove that non-parametric bootstrap methods can produce consistent and second-order correct approximations to the distribution of the Cox statistic for non-nested LISREL-style covariance structure models. We use the method to investigate a question about the specification of a LISREL model used by Kinder, Adams and Gronke (1989). In a second application--a pair of non-nested endogenous switching regression models with tobit-style censoring--we illustrate how bootstrap calibration can be used to correct the size of the test when the test distribution is being estimated by Monte Carlo simulation due to concern about nonregularity.

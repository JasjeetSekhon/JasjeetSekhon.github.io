How do I get RGENOUD to continue a run that was previously terminated?

GENOUD will attempted to continue a previous run if the share.type variable is set equal to 1. This will only work if the previous project file is referenced by the current project.path variable.

For more information, please the write-up of the project.path variable on the manual page.
